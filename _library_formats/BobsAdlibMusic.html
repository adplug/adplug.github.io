Bob's Adlib Music belongs to Official Hamster Republic Role Playing Game Construction Engine, and is one of the now multiple supported music formats in this game engine. Games created using this game engine store there data files in RPG archives. The BAM formatted music files are however renamed when imported into the game to have file-endings in the range .0 - .99.

The BAM file format is documented here.

There exists a tool that can extract BAM files from RPG archives here, and games can be found via this link.
