Softstar Entertainment is a video game developer and publisher, and some of their DOS titles from the 90's used a proprietary OPL music system.

Some games also used a proprietary archive format (*.MKF) that potentially can contain multiple RIX files. File header consists of a series of uint32_t little endian numbers. First number is the header size, the remaining numbers are offset to each individual file - no filenames provided.
